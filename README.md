[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

## Required

- node v12.0.0+

- npm v7.0.0+

- [Docker](https://docs.docker.com/engine/install/) installed

## Installation

```bash
$ npm install
```

## Running the app

```bash
docker-compose up
```

## Test

```bash
# unit tests
$ npm run test

# test coverage
$ npm run test:cov
```
